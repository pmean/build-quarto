::: notes
This package works with RMarkdown to create accessible documents for those individuals who have visual limitations. It provides a headings structure that allows for easy navigation. Note that this package only works in a Windows environment.

Paul J. Northrup. accessr: Command Line Tools to Produce Accessible Documents using R Markdown. R package, 2025-03-30. Available in [html format][ref-northrup-2025].
